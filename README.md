# Game SeaBattle #
Version 1.0 21.12.2015

## Правила игры: ##
1. Корабли расстанавливаются случайным образом
2. Количество кораблей одинаково для всех сторон
3. Размеры кораблей назначаются случайным образом
4. Игру начинает пользователь (Игрок)
